DEFINITION

ML involves teaching a computer to recognize patterns by example, rather than programming it with specific rules. These patterns can be found within data. In other words, ML is about creating algorithms (or a set of rules) that learn complex functions (or patterns) from data and make predictions on it.

WHY ML?
  • Machine learning allows the user to feed a computer algorithm an immense amount of data
  • It allows the computer to analyze and make data-driven recommendations and decisions based on only the input data.
  • It is widely spreaded and we use it in one form or the other.
  • Some of the applications include: Predictions while Computing, Product Recommendations, Search Engine Result Refining, Email Spam and Malware Filtering, Social Media Services, Online Fraud Detection and more.
HOW CAN ML BE USEFUL?

Depending on what one is trying to achieve ML can be used in many form to perform immense data related tasks such as predicting the salary of an employee based on many features/ attributes like skills, experience, demographics and many other parameters. A collegue might use it for predicting career paths and college education option based on the student’s interests and academic performance.
